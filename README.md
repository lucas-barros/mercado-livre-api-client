# Marvel API React Client
Project developed for Aceleradev React. It is a react client application that consumes the Mercado livre API.

## Stack
- React
- Create React App
- Redux
- Redux-saga
- React Router
- Styled-components

## Requirements
- Node.js
- Yarn

## Installation
- Clone this project
- Install with `yarn install`

## Development
- Run `yarn start`
